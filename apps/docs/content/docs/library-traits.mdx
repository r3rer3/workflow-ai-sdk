---
title: Good Workflow Library Traits
description: Events, workflow state, explicit durability, and AI SDK-native message shapes.
---

- It treats events, tools, and model streaming as first-class.
- It exposes native workflow state instead of hiding it behind middleware.
- It makes durability explicit instead of pretending every route can do everything.
- It persists UI messages in the shape the AI SDK already understands.
- It lets teams adopt storage adapters without redesigning their database.
